# SQL and Database Management Through Pop Culture

## Learn Databases with Story-Driven, Hands-On Notebooks

An open, public textbook for SQL and database management, designed for classroom use, self-study, and direct deployment to GitHub Pages. Every chapter is notebook-first and built for immediate practice.

Supported with grant funding from Minnesota State and Rochester Community and Technical College.

## Why This Book

- Public and open educational resource with no paywall
- Built around practical SQL skills and modern database workflows
- Pop culture scenarios that make technical concepts memorable
- Full Google Colab compatibility for zero-setup learning
