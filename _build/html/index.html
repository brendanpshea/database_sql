SQL and Database Management Through Pop Culture#

Open educational resource | Jupyter Book | Google Colab ready

An interactive, public-facing textbook for database design, SQL, data security, performance tuning, and PostgreSQL. The book is built around story-driven notebook chapters that students can read on the web or run immediately in Colab.

Start with Chapter 1 Launch Chapter 1 in Colab

Supported with grant funding from Minnesota State and Rochester Community and Technical College.

At a glance

12 core chapters
2 guided labs
100% web and Colab access
OER public course materials

Built for classroom adoption, independent study, and direct GitHub Pages publishing without requiring local software setup.

Why this book works#

Notebook-first instruction

Each chapter combines explanation, code, and exercises in one place so students can read and practice without context switching.

Pop culture scenarios

Starships, Gotham banks, Mario plumbing, and Pokemon datasets make abstract database ideas easier to remember.

Immediate hands-on practice

Learners can stay in the book for reading or jump directly into Colab for live experimentation with no local install burden.

Career-aligned topics

The sequence covers modeling, querying, security, performance, design, deployment, and professional database workflows.
