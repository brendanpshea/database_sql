SQL and Database Management Through Pop Culture#

Open educational resource | Jupyter Book | Google Colab ready

An interactive, openly licensed textbook for database design, SQL, security, performance, and PostgreSQL. Twelve story-driven chapters read cleanly on the web and run live in Colab — no local install required.

Development of this textbook was supported by grant funding from Minnesota State and Rochester Community and Technical College as part of their commitment to high-quality open educational resources.

Why this book works#

Notebook-first instruction

Each chapter combines explanation, runnable code, and exercises in one place so students read and practice without context switching.

Pop-culture case studies

Starships, Gotham banks, Mario plumbing, and Pokémon datasets make abstract database ideas easier to remember — and a little more fun to teach.

Career-aligned coverage

From modeling and querying through indexes, transactions, security, and PostgreSQL deployment — the topics students actually encounter on the job.

Zero-setup access

Read in the browser or launch any chapter in Colab with one click. No local install, no version mismatches, no lost class time.
